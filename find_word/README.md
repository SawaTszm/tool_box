# Find Word

(名誉の為に言っておくと普段は使わないけど)ネイティブじゃない言語で特定のアルファベット含む5文字の単語を探し出すのって結構大変じゃないか？と思って、それを探す関数を書けそうだなと思ったので試作版(を作る予定)です。

## 現状

正規表現`^(?=.*hoge)(?=.*huga)~以下略~.*$`を使って入力したアルファベットを含む単語を探します。

```zsh
$python find_word.py

# 探したい文字を入れてください: ft
# ['fight']
```
